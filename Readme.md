# Ember

## Made in 2019 IndieGameWeekend
## by Team "Mum - Moom - Mi"

---

## Operating method  
A, D : 좌, 우 이동 ( Lateral move )  
Space : 점프 ( Jump )  
Mouse LClick : 공격 ( Attack )  
Mouse RClick : 마우스 방향 돌진 ( Dash )  
F : 상자 열기, 아이템 획득 ( Open Chest, Get Items )
